• stocks
Table that lists stocks.
    • id INTEGER PRIMARY KEY NOT NULL UNIQUE
Column with an id for each stock
    • name TEXT
common name of the stock
    • symbolgoogle TEXT NOT NULL UNIQUE
Google finance symbol for this stock
    • symbolyahoo TEXT NOT NULL UNIQUE
Yahoo finance symbol for this stock
    • exchangeid INTEGER NOT NULL
id of the exchange where this stock is traded. This id is a reference to the exchanges table.
    • tracked BOOLEAN NOT NULL DEFAULT False
Whether or not to acquire periodical quotes of this stock
    • interval INTEGER DEFAULT 5
Interval in minutes between quotes acquisition, if “tracked”
    • lastquote REAL
Last quote for this stock
    • lastquotestamp DATETIME
Time and date of the last quote for this stock
    • currencyid INTEGER REFERENCES currencies (id), type TEXT
Currency in which this stock is traded. It is a reference for the currencies table
  • strategies
Table that hold several strategies, per stock, to decide when to sell or to buy
    • id INTEGER PRIMARY KEY NOT NULL UNIQUE
Column with an id for each strategy
    • stockid INTEGER REFERENCES stocks (id)
Reference of the stock that is the target of this strategy
    • active BOOLEAN
Whether this strategy is active
    • lowcount INTEGER
Maximum continuous number of days of stock down, before deciding to buy
    • minreturn REAL
Minimum required return (in %) before deciding to sell
  • portfolio
Table that holds our current portfolio of stocks
    • id INTEGER PRIMARY KEY UNIQUE NOT NULL
Column with an id for each element of the portfolio
    • stockid INTEGER REFERENCES stocks (id) NOT NULL UNIQUE
Column that references the stock of this element of the portfolio
    • qty REAL
Quantity on hond of this stock
    • cost REAL
Unit cost for each unit of this stock
  • movements
Table with movements on the fortfolio
    • id INTEGER PRIMARY KEY UNIQUE
Column with an id for each movement
    • stockid INTEGER REFERENCES stocks (id) NOT NULL
Column that references the stock
    • date DATETIME
Date and time of the movement
    • qty REAL
Quantity involved in the movement
    • value REAL
Value of the movement per unit of the stock
    • "action" STRING
Action of the movement. Can be “buy”, “sel” or “dividend”
    • options STRING
To use in the future to list options ofr the movement. For example value of commissions.
  • currencies
Table that holds currencies that the database understands
    • id INTEGER PRIMARY KEY NOT NULL UNIQUE
Column with reference of this currency
    • symbol TEXT NOT NULL UNIQUE
Symbol normally used for this currency. Example: $, €, R$, ...
    • shortname TEXT
Three letter code for this currency
  • exchanges
Table that holds stock exchanges, where stocks are listed
    • id INTEGER PRIMARY KEY NOT NULL UNIQUE
Column with the reference of this stock exchange
    • name TEXT
Common name of the exchange
    • countrycode TEXT
Three letter country code where the exchange is situated
    • shortnamegoogle TEXT
Google Finance code of this exchange
    • shortnameyahoo TEXT
Yahoo Finance code of this exchange
    • openhour TIME
Open time for the exchange. In local time, and in the format hh:mm:ss
    • closehour TIME
Close time for the exchange. In local time, and in the format hh:mm:ss
    • workingdays TEXT
Comma separeted list of the week working days for the exchange. 0 for monday. Example: 0,1,2,3,4
    • comission REAL
Value of commission applied on each sell or buy in this exchange
    • taxondividends REAL
Value of income tax rate applied on each dividend received. For example 0.26
  • dividends
Table that hold a list of historical dividends on each stock
    • id INTEGER PRIMARY KEY NOT NULL UNIQUE
Column that holds reference of this dividend
    • stockid INTEGER REFERENCES stocks (id)
Stock id reference of the stock dividend
    • date DATE
Ex-dividend date for this dividend
    • value DOUBLE
Value of the dividend per share
  • options
Table to list several options for the database
    • id INTEGER PRIMARY KEY NOT NULL UNIQUE
Reference of this option
    • name TEXT
Name of the option
    • value TEXT
Value of the option
  • quotes
Table that holds hostorichal quotes for each stock
    • id INTEGER PRIMARY KEY NOT NULL UNIQUE
Reference of this quote
    • stockid INTEGER NOT NULL REFERENCES currencies (id)
Stock id reference for the quote
    • timestamp DATETIME NOT NULL
Date and time format of the quote (in ISO 8601 format with time zone offset from UTC)
    • value FLOAT NOT NULL
Value of the quote
  • hollidays
Table that holds if a date is a holiday in a particular country/zone
    • id INTEGER PRIMARY KEY UNIQUE NOT NULL
Reference column for this entry in the table
    • country TEXT
Country code (three letter)
    • zone TEXT
Region (not used for the moment)
    • date DATETIME
Date (in the format yyyy-mm-dd)
    • holliday BOOLEAN
Whether it is a holiday or not
